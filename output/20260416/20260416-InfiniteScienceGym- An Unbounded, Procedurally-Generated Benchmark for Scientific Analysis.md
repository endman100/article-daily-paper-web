InfiniteScienceGym: An Unbounded, Procedurally-Generated Benchmark for Scientific Analysis

InfiniteScienceGym：一個無邊界、程序生成的科學分析評測基準

這篇論文提出一個全新科學評測平台，專為測試 LLM 在科學分析上的能力設計

平台能自動生成包含資料夾結構、檔案與表格資料的科學倉庫，並搭配可驗證的問答任務

這種設計讓評估 AI 的證據推理、放棄作答與工具運用能力變得更精確

👇

----
InfiniteScienceGym 不依賴現有的公開數據集，能針對 LLM 難以處理的盲點與失誤進行測試

透過程序生成，平台可無限擴展題庫，避免傳統資料集的出版偏誤與知識侷限

每個科學倉庫都能根據種子值重現，確保評測的公平性與可驗證性

這種方式也大幅減少儲存空間需求，提升評測效率

讓 AI 評測更貼近真實科學挑戰

----
平台設計了一套問答生成器，能產生可回答與不可回答的問題，並給出精確標準答案

這有助於檢驗 LLM 是否能正確識別無法回答的問題，避免亂猜

同時也能測試模型在工具輔助下的分析能力，而不只是單純依賴語料庫

這對於推動 AI 在科學領域的應用具有重要意義

研究者可針對不同模型進行細緻比較

----
論文發現，目前無論是開源還是專有模型，在這個平台上的整體正確率都不高

辨識不可回答問題仍是 LLM 的主要弱點，顯示現有模型在科學推理上還有很大進步空間

較強的模型會更善用工具，而不是單純增加推理步驟或語料量

這反映出工具運用能力對於提升 AI 科學分析表現至關重要

平台有助於揭示模型的真實能力

----
InfiniteScienceGym 補足了現有科學評測的不足，能針對難以用公開數據集評估的情境進行測試

研究者可利用這個平台設計多樣化的科學任務，全面檢驗 LLM 的推理與分析能力

未來，這類程序生成的評測方式有望成為 AI 科學研究的新標準
