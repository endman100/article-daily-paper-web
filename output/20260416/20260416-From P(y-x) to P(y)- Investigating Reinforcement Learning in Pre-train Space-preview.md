From P(y|x) to P(y): Investigating Reinforcement Learning in Pre-train Space

從條件分佈到邊際分佈，探索強化學習在預訓練空間的潛力

當前主流的 LLM 推理增強方式，是在後訓練階段用帶有可驗證獎勵的強化學習（RLVR）去優化條件分佈 P(y|x)，也就是給定問題、生成答案的訓練方式

但這個方法有一道根本性天花板，訓練效果終究被基礎模型原本的輸出分佈所限制

這篇研究換了一個角度，直接在預訓練空間優化邊際分佈 P(y)，移除問題條件的束縛，讓獎勵驅動的更新直接作用在模型的內在分佈上

研究最終提出了 DSRL（Dual Space RL），透過兩段式訓練策略，在多項推理基準上超越了過去的強基準

那麼，拿掉問題條件來訓練 LLM，真的能學到推理能力嗎 👇

----
要理解這篇研究的出發點，先要搞清楚標準 RLVR 在優化什麼

RLVR 在後訓練階段透過讓模型回答問題、並給予正確與否的獎勵訊號，讓輸出分佈往高獎勵方向移動

這個過程本質上是在優化條件分佈 P(y|x)，也就是給定問題 x 時，生成答案 y 的機率

這種訓練路線行之有效，但有研究發現，模型能學到的推理路徑，其實早已隱含在基礎模型的輸出分佈之中

換句話說，後訓練 RL 能帶來的進步空間，很大程度取決於基礎模型本身的探索空間有多廣

如果基礎模型從一開始就沒有某些推理路徑，後訓練 RL 再努力，也只能在有限的分佈內打轉

這個發現促使研究者思考，能不能直接在預訓練空間就先拓展這個推理基礎

----
PreRL（Pre-train Space RL）的核心改動很直接，就是移除更新時的問題條件 x

標準 RL 每次更新時，梯度計算依賴 log P(y|x)，也就是給定問題 x 的條件下生成答案 y 的對數機率

PreRL 改為計算 log P(y)，直接優化邊際分佈，把問題 x 從計算中移除

這個改動有一個需要回答的根本問題，拿掉問題條件之後，梯度更新方向還能和標準 RL 保持一致嗎

研究從理論推導和實驗觀測兩個層面驗證，log P(y) 和 log P(y|x) 的梯度之間有很強的對齊關係

這個對齊性說明 PreRL 確實是標準 RL 的一個有效替代路線，而且天然保留了模型更廣的探索空間

預訓練空間的學習讓推理能力直接被編碼進模型參數，打下更紮實的推理基礎

----
在拆解 PreRL 的具體效果時，研究者發現了正樣本強化和負樣本強化之間的一個關鍵不對稱性

正樣本強化（PSR）在預訓練空間中反而讓性能下滑，原因是模型把機率質量堆積在自己生成的正確樣本上，收窄了後續探索的範圍

相較之下，負樣本強化（NSR）的表現出乎預期，成為推理提升的核心驅動力

NSR-PreRL 會快速修剪預訓練空間中的錯誤推理路徑，幫模型清理掉不正確的推理方向

在這個修剪過程中，模型也開始自發地產生更多過渡性思考和反思行為，讓推理過程變得更豐富

這些反思行為的增長幅度相當可觀，說明 NSR-PreRL 能從模型內部激發出潛在的推理能力

這種由內而外的激發方式，跟外部強行約束推理格式的做法完全不同

----
基於 NSR-PreRL 的發現，研究者提出了最終的訓練策略 DSRL（Dual Space RL）

DSRL 採用策略再生（Policy Reincarnation）的概念，把整個訓練拆成兩個連續階段

第一階段用 NSR-PreRL 初始化模型，目標是在預訓練空間拓展推理視野，剪除錯誤路徑，激發內在推理能力

第二階段切換到標準 RL 進行條件分佈的精細優化，讓模型在已拓展的探索空間內做更精準的調整

NSR-PreRL 先清理不正確的推理方向，標準 RL 再在乾淨的空間中精調，兩個階段各司其職

實驗結果顯示，DSRL 在多個推理基準上持續超越強基準，樣本使用效率也更高

Pass@K 指標的提升說明 DSRL 確實把模型的推理探索空間導向了更正確的子空間

----
這項研究指出了一個長期被忽視的訓練維度，預訓練空間本身也可以成為主動優化的目標

過去的訓練邏輯通常把預訓練和後訓練 RL 視為兩個分開的流程，各自負責不同的學習目標

但 PreRL 顯示，在預訓練空間注入獎勵訊號是可行的，梯度方向能和後訓練 RL 保持一致

NSR 機制的發現也提供了一個反直覺的洞察，負樣本的訓練價值有時候比正樣本更難被看見，效果卻更好

如果推理能力的天花板並非由後訓練 RL 決定，而是由預訓練空間的廣度決定，訓練資源的配置方式或許值得重新思考

DSRL 的兩段式框架，為未來 LLM 訓練流程的設計提供了一個值得借鑑的思路
