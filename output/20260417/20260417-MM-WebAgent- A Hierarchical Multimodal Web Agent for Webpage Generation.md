# MM-WebAgent: A Hierarchical Multimodal Web Agent for Webpage Generation

MM-WebAgent
為網頁生成而生的分層多模態網頁代理人
AIGC 讓圖片、影片、視覺元素能按需生成
這對現代 UI/UX 是很大的助力
但把這些工具直接串到網頁自動生成裡，結果常常很尷尬
各元素單獨生成，風格不一致，整體協調性很差
MM-WebAgent 想用分層代理人來解這個問題
👇

---

整體架構是分層規劃加反覆自我反思
上層負責整體版面與全局風格決策
下層負責具體多模態內容生成
中間還有一道整合層，確保不同元素相互協調
這種設計讓局部優秀與整體一致同時被照顧到
而不是只選其中一個

---

AIGC 元素在生成時不是單點行動
代理人會根據上層的語意與風格決策給出條件
每個元素都被放在整體脈絡裡做
如果生成結果不協調，自我反思機制會啟動
讓代理人重新調整參數或重試
這種迴圈式生產提升了最終品質

---

為了系統化評估，作者建立了一個多模態網頁生成基準
搭配多層次的評估協定
從版面、內容、整合三個角度分別打分
這在這個領域是少見的細緻標準
過去研究常常只用主觀感受
現在比較變得可重複、可量化

---

實驗顯示 MM-WebAgent 在多模態元素的生成與整合上都勝過代碼生成基線與其他代理人基線
尤其在跨元素一致性和整體視覺協調上差距明顯
這印證了分層規劃加反思的設計思路
不是靠某一個強大元件，而是靠整體協作
這種思維移植到其他多媒體生成領域也適用

---

MM-WebAgent 體現了代理人設計的一個新階段
當任務是多模態、多元素、全局一致時
單一大模型很難一次搞定
分層代理人把決策與執行拆開
讓每一層只需處理它能處理的事
這種軟體工程式的思維會越來越重要
