LLM 能扮演小說角色跟你聊天，但它真的懂那個角色嗎？

角色扮演對話 Agent（RPCA）是近年 LLM 應用中極受歡迎的方向，從虛擬偶像到情感陪伴都有它的身影

然而長期以來，這類 Agent 缺乏一個全面的評測基準，導致大家各說各話，難以比較好壞

CharacterEval 就是為了解決這個問題而誕生的，一個專為中文角色扮演對話設計的完整 Benchmark

它到底怎麼評，又發現了什麼有趣的結論？

👇

----
CharacterEval 的資料來源非常講究，所有角色都取材自中文小說和劇本

總共收錄了 77 個不同的角色，涵蓋各種性格，身份與故事背景

整個資料集包含 1785 段多輪對話，加起來超過 23K 個對話範例

這些對話並非隨意生成，而是先由 GPT-4 進行初步提取，再經過人工嚴格品質把關

每個角色還搭配了從百度百科蒐集的深度人物檔案，確保背景資訊完整豐富

----
評測設計是 CharacterEval 的一大亮點，採用了多面向的評估架構

它將角色扮演能力拆分為四大維度，從不同角度去審視 Agent 的表現

總共設計了 13 項具體指標，分別衡量角色的知識掌握，語言風格，情感表達和行為一致性

這套評測框架的好處在於，它不只看像不像，還看懂不懂和穩不穩

透過如此精細的拆解，研究者能更準確地找出 Agent 在角色扮演中的弱項

----
從四個維度來看，第一個維度關注角色知識的掌握程度

比方說，Agent 是否記得角色的重要經歷，人際關係，甚至口頭禪

第二個維度評估語言風格的還原度，每個角色說話的語氣和用詞習慣是否到位

第三個維度則是情感表達，角色在特定場景下是否展現出合理的情緒反應

最後一個維度是行為一致性，也就是 Agent 是否能在長對話中始終維持同一個角色的人設

----
實驗結果揭示了一個有趣的現象

在中文角色扮演場景中，專門針對中文訓練的 LLM 反而展現出比 GPT-4 更強的表現

這說明角色扮演任務高度依賴語言和文化的理解深度，而非單純的模型規模

中文小說裡的角色有大量文化背景和語境暗示，中文原生模型在這方面有天然優勢

另一方面，所有模型在長對話中維持角色一致性這件事上，都還有很大的進步空間

----
CharacterEval 的出現，為中文角色扮演 Agent 的研究提供了一個統一的比較平台

過去各團隊用不同資料，不同指標來測試，結果根本無法互相比較

現在有了標準化的 Benchmark，研究者可以更有效率地推進這個領域的發展

它也提醒我們，角色扮演絕非只是回答問題這麼簡單，而是需要深層的文化理解與情感模擬

當 AI 越來越常被用於陪伴型應用時，如何確保它真的理解角色而非只是模仿，是值得持續思考的課題
