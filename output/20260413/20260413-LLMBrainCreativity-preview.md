Large Language Models Align with the Human Brain during Creative Thinking

LLM 在創造力測試上的表現越來越好，但這種創造力跟人腦的創造力有關聯嗎

這篇研究用 fMRI 腦造影資料比對了 LLM 的內部表示和人類創意思考時的腦部活動

結果發現兩者之間確實存在有意義的對齊，而且對齊程度受模型大小和訓練方式影響

AI 的創造力居然跟人腦有結構性的相似，這背後有什麼故事 👇

----
研究團隊用了 170 位受試者進行替代用途測試 AUT 時的 fMRI 資料

AUT 是一個經典的發散思考測試，要求受試者想出一個物品的各種非常規用法

研究者從不同大小的 LLM 中提取內部表示，範圍從 270M 到 72B 參數

再用 Representational Similarity Analysis 來量測 LLM 表示和腦部活動的對齊程度

分析聚焦在跟創造力相關的兩個腦網路：default mode network 和 frontoparietal network

----
第一個發現是腦 - LLM 對齊會隨模型大小增加而提升

這個效果在 default mode network 上特別明顯

第二個發現是對齊程度跟想法的原創性有關

越有創意的想法，對應的 LLM 表示和腦部活動越相似

這種效果在創意過程的早期階段最強

----
更有趣的是不同後訓練目標會以不同方式影響對齊

經過創意優化的 instruct 模型保持了跟高創意腦部反應的對齊，同時降低了跟低創意的對齊

經過人類行為微調的模型則同時提升了跟高創意和低創意腦部反應的對齊

但經過推理訓練的模型反而降低了對齊，把表示從創造性的神經幾何推向了分析性處理

這說明 chain-of-thought 訓練可能在內部層面把模型從創造性模式引開了

----
這些結果表明後訓練目標會選擇性地重塑 LLM 的內部表示

這種重塑跟人類創意思考的神經結構之間存在可量測的對應關係

AI 的創造力不只是表面上像人類，在內部表示的層面上也有結構性的相似

這為理解 AI 創造力的本質提供了一個神經科學的視角
