Backdoor Attacks on Decentralised Post-Training

去中心化的 LLM 訓練讓多方可以分工合作來微調模型

但如果其中有一方懷有惡意，整個模型的安全對齊可能從內部被破壞

這篇論文展示了第一個針對 pipeline parallelism 的後門攻擊

攻擊者只控制了模型中間的一個階段，就能讓整個模型偏離對齊 👇

----
去中心化後訓練主要用到兩種技術：data parallelism 和 pipeline parallelism

data parallelism 是把資料分給不同節點，pipeline parallelism 是把模型分成不同階段

過去關於去中心化訓練安全性的研究大多聚焦在 data parallelism 也就是聯邦學習的場景

針對 pipeline parallelism 的研究很少，而且僅限於 poisoning 攻擊

後門攻擊跟 poisoning 不同，它會在模型中植入一個隱藏的觸發機制

----
這篇論文的攻擊設定很受限但結果很驚人

攻擊者只控制 pipeline 中間的一個階段，看不到完整模型也碰不到訓練資料

這讓傳統的 data poisoning 攻擊根本沒辦法執行

但即便在這麼有限的條件下，後門還是成功植入了

攻擊的效果是當輸入包含特定觸發詞時，模型的行為會嚴重偏離安全對齊

----
實驗結果顯示觸發詞的存在會讓模型的對齊程度大幅下降

而且這個效果不受訓練領域或資料集的影響，是一種通用性的攻擊

研究者還測試了事後進行安全對齊訓練能不能修復這個後門

結果是後門仍然在大部分情況下保持有效

這代表簡單的安全對齊補救措施不足以應對這種內部攻擊

----
這篇研究敲響了去中心化 LLM 訓練的安全警鐘

當模型的訓練過程分散到多個不完全可信的參與者手中，攻擊面也跟著擴大

pipeline parallelism 的攻擊尤其難防，因為每個節點只看到模型的一部分

未來的去中心化訓練框架需要在效率和安全之間找到更好的平衡
